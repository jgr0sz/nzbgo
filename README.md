# nzbgo
[![Go Reference](https://pkg.go.dev/badge/github.com/jgr0sz/nzbgo.svg)](https://pkg.go.dev/github.com/jgr0sz/nzbgo)

Comprehensive, spec-compliant NZB parser and meta editor library implemented in Golang.

NZB files, or Newzbin files, are XML-based files used for the transfer of content from Usenet servers to the end user by providing the necessary specifications for a client to reconstruct files. More information can be found on the specifications of this filetype and Usenet as a whole [here](https://sabnzbd.org/wiki/extra/nzb-spec).
